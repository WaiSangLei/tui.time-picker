    inputType string <optional>
    		 'selectbox'

    showMeridiem boolean <optional>
    		 true

    Show meridiem expression?
    meridiemPosition string <optional>
    		 'right'

    Set location of the meridiem element. If this option set 'left', the meridiem element is created in front of the hour element.
    language string <optional>
    		 'en'

    Set locale texts
    usageStatistics Boolean <optional>
    		 true|false

    send hostname to google analytics [default value is true]
    Example
    var timepicker = new tui.TimePicker('#timepicker-container', {
    initialHour: 15,
    initialMinute: 13,
    inputType: 'selectbox',
    showMeridiem: false
});
